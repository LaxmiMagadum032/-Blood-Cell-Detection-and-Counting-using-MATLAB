# -Blood-Cell-Detection-and-Counting-using-MATLAB

This project focuses on the detection and counting of **Red Blood Cells (RBCs)**,  
**White Blood Cells (WBCs)**, and **Platelets** using **Digital Image Processing techniques**.

The methodology is inspired by the IEEE paper  
**"Implementation of Blood Cell Counting Algorithm using Digital Image Processing Techniques"**.
